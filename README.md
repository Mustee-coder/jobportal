# Job Portal

A full-stack Job Portal application built with the MERN Stack. The platform connects job seekers with employers, allowing users to search and apply for jobs while employers can post and manage job listings through a secure dashboard.

## 🚀 Features

### Job Seekers
- User registration and login
- Browse available jobs
- Search and filter jobs
- View job details
- Apply for jobs
- Manage profile

### Employers
- Secure authentication
- Post new jobs
- Edit and delete job listings
- View applicants
- Manage posted jobs

### Admin
- Dashboard overview
- Manage users
- Manage employers
- Manage job listings
- Platform monitoring

## 🛠 Tech Stack

### Frontend
- React
- React Router
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Bcrypt
